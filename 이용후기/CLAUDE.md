# 이용후기
> 카테고리: **클라이언트 (Client)** — `~/.claude/CLAUDE.md` 글로벌 규칙 + 카테고리 C 규칙 상속

## 목적
클라이언트 A 프로젝트 — 이용후기 콘텐츠 제작 및 카드뉴스 변환

## NDA 준수 사항
- 클라이언트명 외부 노출 금지
- 내부 수치·데이터 공개 금지
- 작업물 타 프로젝트 유용 금지
- 커밋 메시지에 클라이언트명 포함 금지

## 클라이언트 A 브랜드 톤
- 신뢰·전문성 중심 어조
- 고객 목소리 인용 시 원문 최대한 유지
- 과장·미화 없이 사실 기반 편집

## 카드뉴스 자동생성 연결
- 이용후기 → 카드뉴스 변환 파이프라인 사용
- 입력: 고객 후기 텍스트 → `CardNewsInput.body`에 매핑
- 플랫폼: `instagram-post`, `kakao` (클라이언트 승인 채널만)
- 납품 전 클라이언트 시안 확인 필수

## 파일 관리
- 납품용 최종본: `./final/` 폴더
- 작업 중 파일: `./wip/` 폴더
- 클라이언트 원본 자료: `./source/` (수정 금지)

## 금지사항
- 🚫 클라이언트 내부 자료 외부 공유
- 🚫 `./source/` 폴더 내 파일 수정
- 🚫 납품 전 클라이언트 미승인 배포
- 🚫 타 클라이언트 스타일 혼용
