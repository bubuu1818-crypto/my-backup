구체적인 자동화 전략
1. 템플릿 기반 시스템
미리 정의된 레이아웃 템플릿을 만들어 두세요:

타입 A: 큰 이미지 + 텍스트 (50:50 비율)
타입 B: 이미지 중심 + 작은 텍스트 (70:30)
타입 C: 그라데이션 배경 + 중앙 텍스트

2. 타이포그래피 자동 조정

제목 글자수 초과 시 자동으로 폰트 크기 줄이기
본문은 최대 3줄 강제 (초과분 생략 기호 처리)
가독성 확보: 텍스트 배경에 50~70% 투명도 검정 오버레이

3. 컬러 팔레트 체계화
브랜드 메인 색상 → HSL 기반 자동 변형
  ├─ 주색 (100% saturation)
  ├─ 보조색 (70% saturation) 
  └─ 배경색 (20% saturation)
4. 안전영역 마진

좌우: 60px (디바이스 곡률 고려)
상하: 80px (상단 노치, 하단 홈 인디케이터)
텍스트: 안전영역 내에만 배치

5. 이미지 최적화

자동 크롭: 중요 영역 중심으로 (얼굴 감지 활용)
포맷: WebP 우선, 폴백 JPG
크기: 모바일 기준 1-2MB, 데스크톱은 최대 3MB

6. 동적 요소 규격

페이지 번호: 하단 우측 고정
진행률 바: 상단 풀너비 (2-3px)
CTA 버튼: 높이 48-56px (터치 기준)
