# 📄 디자인 브리프 I: 종합 보고서 (Comprehensive Report) 시스템 표준화

**✨ 목표:** 서비스가 단순 정보를 제공하는 것을 넘어, 시장 및 산업에 대한 깊이 있는 통찰력과 전문 지식을 보유한 **'신뢰할 수 있는 출처(Authority Source)'**임을 시각적으로 증명한다.
**🎯 톤앤매너:** 학술적, 데이터 기반, 미니멀리즘, 신뢰감 (Academic, Data-Driven, Minimalist)

## 🎨 비주얼 시스템 스펙

| 항목 | 사양/규칙 | 상세 내용 | 참고 레퍼런스 |
| :--- | :--- | :--- | :--- |
| **전체 그리드** | 12-컬럼 반응형 그리드 시스템 채택 (Desktop 기준) | 가독성을 극대화하기 위해 콘텐츠를 세로/가로 축으로 균일하게 분할. | 신뢰도 높은 금융 보고서, 컨설팅사 PPT 스타일 |
| **주요 색상 (Primary)** | `#0D2A5F` (딥 네이비) | 헤더, 주요 제목(H1), 강조 박스 배경색. 전문성과 깊이를 상징. | - |
| **보조 색상 (Secondary)** | `#B3CDE0` (라이트 스카이 블루) | 그래프의 기본 요소, 인용구 블록 배경. 지적이고 개방적인 느낌 부여. | - |
| **강조색 (Accent/CTA)** | `#FF6F47` (따뜻한 오렌지) | 핵심 결론, KPI 값 하이라이트, '자세히 보기' CTA 버튼. 시선 집중 효과 극대화. | - |
| **폰트 조합** | 1. 제목: Pretendard Bold/SemiBold<br>2. 본문: Noto Sans KR Regular/Medium | 가독성이 높고, 디지털 환경에서 권위적인 느낌을 주는 산세리프 계열로 통일. | (적정 크기: H1 - 36pt, Subtitle - 20pt, Body - 14pt) |

## 📊 데이터 시각화 및 레이아웃 표준

1.  **데이터 테이블:**
    *   **규칙:** 행(Row)마다 배경색을 미묘하게 다르게 처리하여 가독성을 확보 (Zebra Striping).
    *   **핵심:** 값(Value) 부분에만 강조색(`Accent`)을 사용하여 주목도를 높인다.
2.  **인용구 블록 (Quote Block):**
    *   **레이아웃:** 딥 네이비 배경의 왼쪽에 세로 점선 또는 큰 따옴표 아이콘 배치. 본문 내용보다 간격(Padding)을 충분히 주어 독립적인 정보 단위임을 강조한다.
3.  **그래프 및 차트:**
    *   **원칙:** 데이터의 흐름과 변화를 명확하게 보여주는 **꺾은선 그래프 (Line Chart)**와 비교 분석에 용이한 **막대 그래프 (Bar Chart)** 위주로 사용을 제한한다. 복잡한 3D 효과는 금지한다.

# 🔗 디자인 브리프 II: Authority Contact Widget 통합 스펙

**✨ 목표:** 모든 핵심 콘텐츠 페이지에서 자연스럽게 발견되면서도, 클릭 시 **'이것이 전문적인 연락처다'**라는 인식을 심어주는 간결하고 권위적인 위젯을 구현한다.
**🎯 톤앤매너:** 즉각적, 명료함, 접근성 (Instant, Clear, Accessible)

## 📐 스펙 상세 정의 및 통합 가이드

### 1. 기본 레이아웃 (Default State)
*   **크기:** 권장 너비: 280px / 높이: 70px. (모바일 환경에서는 플렉스박스를 이용해 페이지 폭의 90%로 자동 축소).
*   **배경:** `#FFFFFF` (흰색). 배경에 그림자 효과(Shadow)를 주어, 콘텐츠와 분리된 **'독립적인 모듈'**처럼 보이게 한다.
*   **구조:** 왼쪽 정렬 (Left-aligned)을 기본으로 하며, 폰트 크기와 간격을 통해 명확한 계층 구조를 만든다.

### 2. 요소별 디자인 스펙
| 요소 | 내용물 | 스타일/스펙 | 목적 |
| :--- | :--- | :--- | :--- |
| **헤더 타이틀** | "전문 상담 문의" 또는 "비즈니스 문의 (Authority Contact)" | Noto Sans KR Bold, 16pt. 색상: `#0D2A5F`. | 위젯의 목적을 명확히 정의하여 신뢰도를 높인다. |
| **전화번호 영역** | `📞 02-6494-0119` | **가장 크게 강조.** Pretendard SemiBold, **24pt**. 색상: `#FF6F47` (강조색). <br>**필수:** 전용 `<a href="tel:0264940119">` 태그 적용. | 가장 중요한 CTA(Call-to-Action)이므로 크기와 대비를 극대화한다. |
| **보조 텍스트** | "평일 오전 9시 - 오후 6시 (월~금)" | Noto Sans KR Regular, 12pt. 색상: `#777777`. | 이용 가능 시간을 명확히 하여 사용자의 기대를 관리한다. |

### 3. 상호작용 스펙 (Interaction States)
*   **Hover State (마우스 오버):** 위젯 전체 배경에 투명도 5%의 연한 회색 오버레이를 적용하고, 전화번호 색상을 `#0D2A5F`에서 `#FF6F47`로 미세하게 '펄스' 효과(Pulse)를 주어 상호작용 가능함을 알려준다.
*   **Mobile Adaptation:** 화면 크기가 768px 이하일 경우, 위젯의 너비를 페이지 폭 대비 최대 90%로 조정하고, 모든 요소 간의 Vertical Padding을 증가시켜 터치하기 쉽게 만든다.
