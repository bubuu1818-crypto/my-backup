## 📞 Call Us Widget 목표 페이지 디자인 시안 3가지 제안 (UX 기반)

CEO님의 지시에 따라, 기존 브랜드의 전문성과 신뢰감('모두의병원동행')을 유지하면서도, 사용자에게 가장 높은 '전화 버튼 클릭률(CTR)'과 낮은 '인지적 마찰력'을 제공하는 디자인 시안 3가지를 제안합니다. 각 시안은 다른 심리적 접근 방식을 취하며, 설치 위치와 목적에 따라 선택할 수 있습니다.

---

### ✨ 전제 조건 및 공통 요소 (Common Elements)
1.  **톤앤매너 유지:** 모든 시안은 '신뢰성(Trust)'과 '전문성(Professionalism)', 그리고 '따뜻함(Warmth)'을 핵심 가치로 합니다.
2.  **폰트:** 본문에는 가독성이 높은 시스템 폰트 (예: Noto Sans KR, Pretendard)를 사용합니다.
3.  **최소화 원칙:** 정보를 최소한으로 전달하며 오직 '전화를 걸도록' 유도하는 것이 목표입니다.

### 💡 시안 1: [Trust Builder] - 신뢰 기반의 전문적 접근 (권장 기본형)
*   **목표 사용자:** 정보 탐색 단계에 있는 잠재 고객 (가장 안정적인 디자인).
*   **UX 전략:** 공식 기관 웹사이트와 같은 안정감과 높은 가독성을 제공하여, 서비스의 신뢰도를 가장 먼저 각인시킵니다.

| 요소 | 상세 스펙 및 코드 | UX/심리적 근거 |
| :--- | :--- | :--- |
| **배경색 (Background)** | `#FFFFFF` (순백색) 또는 아주 연한 아이보리 계열 (`#F9F8F6`) | 깨끗함, 전문성. 복잡한 시선 분산을 막아 안정감을 줍니다. |
| **주요 액션 버튼 색상** | **딥 네이비/블루:** `#004D7A` (신뢰의 색) | 의료 및 IT 분야에서 가장 신뢰도가 높은 블루 계열을 사용하여, 서비스 자체에 대한 확신을 심어줍니다. |
| **보조 강조색 (Icon)** | `#FFC107` (따뜻한 옐로우/골드) | 전화 아이콘이나 '긴급' 문구 등 중요한 포인트를 부각할 때만 사용합니다. 너무 강하지 않게, 주의를 환기시키는 역할을 합니다. |
| **버튼 형태 및 크기** | 모서리 둥글기 (Border Radius): `12px` (적당히 둥근 직사각형). 높이: `56px`. 너비: 부모 요소에 맞춤 (`100%`). | 적당한 곡률은 딱딱함을 완화시키고, 충분한 크기는 모바일 터치 영역을 확보하여 사용성을 극대화합니다. |
| **레이아웃/배치** | 페이지의 하단(Sticky Footer) 혹은 문의 섹션 최상단에 배치. *[전화 아이콘] + [텍스트 설명: 24시간 상담 가능]* → **[딥 네이비 버튼: 전화 걸기]** 순서로 계층적 정보 제공. |
| **핵심 문구 (CTA)** | "지금, 가장 궁금한 점을 전문가에게 문의하세요." (신뢰 기반의 질문 유도) |

### 💡 시안 2: [Warm Helper] - 공감과 편안함을 주는 접근 (Emotional Touch)
*   **목표 사용자:** 보호자 등 심리적으로 불안정하거나 지친 사용자 (따뜻한 위로가 필요한 경우).
*   **UX 전략:** 딱딱한 의료 이미지를 탈피하고, 마치 사람이 직접 말을 건네는 듯한 부드러운 느낌을 주어심리적 장벽을 낮춥니다.

| 요소 | 상세 스펙 및 코드 | UX/심리적 근거 |
| :--- | :--- | :--- |
| **배경색 (Background)** | `#E8F5E9` (매우 연한 민트 또는 페일 그린). | '치유(Healing)'의 느낌을 주는 자연 친화적인 색상으로, 심리적 안정감을 제공합니다. |
| **주요 액션 버튼 색상** | **따뜻한 코랄/테라코타:** `#FF7043` (부드러운 주황빛). | '긴급'함보다는 '돌봄'과 '관심'을 상징하는 온기를 담았습니다. 차분하면서도 시선이 가는 액션 컬러입니다. |
| **보조 강조색 (Icon)** | `#689F38` (차분한 녹색). | 희망, 성장, 안정을 의미하여 전체적인 '케어' 테마를 유지합니다. |
| **버튼 형태 및 크기** | 모서리 둥글기: `20px` (매우 부드러운 곡률). 높이: `60px`. 너비: 콘텐츠 폭에 맞춤. | 높은 곡률은 인간적인 느낌을 강화하며, 사용자가 편안하게 손을 대고 누를 수 있도록 유도합니다. |
| **레이아웃/배치** | 페이지 중앙의 주요 콘텐츠 섹션 끝이나, 후기(Testimonial) 영역 바로 밑에 배치. *[아이콘: 💬]* → *[텍스트 설명: 언제든 편안하게 이야기하세요.]* → **[코랄 버튼: 상담 전화 연결]** 순서로 배치를 유도합니다. |
| **핵심 문구 (CTA)** | "어떤 질문이든 괜찮습니다. 저희가 함께 답을 찾아드릴게요." (공감 및 수용적 태도) |

### 💡 시안 3: [Urgent Action] - 즉각적인 행동 유도형 (High Conversion Focus)
*   **목표 사용자:** 지금 당장 문제를 해결해야 하거나, 필요한 정보를 급하게 찾는 사용자 (긴급 상황에 가까움).
*   **UX 전략:** 다른 정보에 방해받지 않도록 시선을 강하게 끌어당기며, '지금 클릭하지 않으면 안 될 것 같은' 인식을 심어줍니다.

| 요소 | 상세 스펙 및 코드 | UX/심리적 근거 |
| :--- | :--- | :--- |
| **배경색 (Background)** | `#FFFFFF` + 미세한 그라데이션 효과를 주어 깊이감을 부여합니다. | 배경 자체는 깔끔하게 유지하되, 버튼과 위젯에만 포커스를 집중시킵니다. |
| **주요 액션 버튼 색상** | **강조 오렌지:** `#FF6F00` (밝고 경쾌하며 시선이 즉각적으로 끌리는 주황색). | '위험'의 빨간색을 피하고, '주의'와 '행동 개시'를 의미하는 오렌지를 사용하여 긴급성을 부여하되, 신뢰도는 유지합니다. |
| **보조 강조색 (Icon)** | `#FF6F00`과 동일하게 사용하거나, 전화 아이콘에만 짙은 검정/네이비를 사용하여 대비 효과를 극대화합니다. | 버튼 색상 자체로 시선 집중을 유도하며, 반복적인 컬러 코드는 일관성을 높입니다. |
| **버튼 형태 및 크기** | 모서리 둥글기: `8px` (직사각형에 가까워 신뢰감을 유지). 높이: `50px`. 너비: 화면 폭의 약 90%를 차지하는 *고정된 위젯* 형태로 설계. | 직각에 가까운 형태는 단호하고 즉각적인 행동을 요구할 때 효과적입니다. 고정 위젯은 스크롤 위치와 무관하게 항상 노출되어야 합니다. |
| **레이아웃/배치** | 웹페이지 상단(Sticky Header) 또는 하단(Sticky Footer)에 작은 배너 형태로 상시 배치. *[전화 아이콘 + 텍스트: 🚨 지금 바로 전문가 상담]* → **[오렌지 버튼: 전화 걸기]** 순서로 구성. |
| **핵심 문구 (CTA)** | "📞 24시간 대기 중입니다. 언제든지 문의하세요." (접근 용이성 강조) |
