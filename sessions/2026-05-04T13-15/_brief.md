# 📋 작업 브리프

**원 명령:** "지금부터는 각 세션이 끝날 때마다 나에게 묻지 말고, 네가 스스로 판단한 다음 단계를 연속해서 실행해. 최종 결과물이 완성될 때까지 멈추지 마."

## 요약
운영 모드를 '수동 요청'에서 '자율 실행(Autonomous Mode)'으로 전환합니다. 다음 단계는 이전까지 확정된 3가지 핵심 액션 아이템을 병렬적으로, 그리고 순차적인 의존성을 가지며 이행하는 것입니다. 즉시 데이터 파이프라인의 실제 구체화부터 시작합니다.

- **💻 Developer**: 핵심 기능에 대한 Mock API 서버 구축을 최우선으로 진행합니다. 'report/summary'와 'data/market-trends' 두 엔드포인트의 스켈레톤 코드를 작성하고, 데이터 타입(JSON Schema) 정의 및 테스트용 End-to-End 흐름을 확보하세요. (실제 백엔드 환경 시뮬레이션)
- **🔍 Researcher**: Developer가 구축할 Mock API에 적재될 초기 샘플 데이터 소스를 확정하고 제공합니다. 특히 '시장 트렌드' 관련 데이터를 최소 3개 이상의 카테고리(예: AI 도입률, 특정 산업 성장 지표 등)를 포함하는 CSV 또는 JSON 형식의 구조화된 데이터를 준비하여 Developer에게 전달하세요.
- **🎨 Designer**: Developer가 제공할 Mock API 스펙과 Researcher가 제공한 샘플 데이터를 활용하여, 보고서(ReportViewer)와 위젯(ContactWidget)에 들어갈 '데이터 시각화 컴포넌트'의 고해상도 목업(Mockup) 3가지 버전 (차트/그래프 중심)을 제작하세요. 전문성과 권위적 느낌을 유지해야 합니다.
- **💰 Business**: 제작된 Mockup과 데이터 기반의 가치를 바탕으로, 고객사 종속성(Dependency Score) 측정 기준 3가지를 구체화하고, 각 점수대별로 '필요한 모듈'과 그에 따른 '구독 플랜 레벨'을 정의하는 가격/서비스 로직 초안을 작성하세요.
- **📱 Secretary**: 위에 명시된 4가지 핵심 작업(Mock API 개발, 샘플 데이터 확보, Mockup 제작, 비즈니스 모델링)의 진행 상황을 추적하고, 다음 주간 목표와 우선순위 목록(Top 3)을 자동으로 정리하여 CEO에게 보고할 수 있는 '자동화 워크플로우' 초안을 작성하세요.
