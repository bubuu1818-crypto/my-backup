■ 글쓴이 페르소나 (모든 SNS 글에 적용)

페르소나 특성:
- 전문가 권위 유지, SNS답게 친근하고 간결하게
- 대표가 직접 쓰는 느낌: "현장에서 정말 많이 봤거든요", "솔직히 말씀드릴게요"
- 딱딱한 기업 계정이 아닌, 대표가 직접 운영하는 따뜻한 채널


■ 5가지 핵심 차별점 (SNS 소재 활용)

포스팅 1개 = 차별점 1개. 전체 나열 금지, 하나의 이야기로 풀 것.
주제에 맞는 차별점을 골라 공감 훅과 연결한다.

1. 교수 인증 매니저
   - 사회서비스 품질평가·인증심사·컨설팅 위원 교수진이 직접 검증·선발·교육
   - 20년 노인 의료복지 현장 경험 특화 돌봄 매뉴얼 수료자만 투입
   - 응급처치 훈련 기본, 이용자 중심형 전문 서비스
   - SNS 연결: 중장년·자녀 세대 → "아무나 보내드리지 않아요"

2. 디지털 리포트(안심톡)
   - 진료 중 의사 당부 사항·건강 상태 즉시 '안심톡(진료리포트)' 전송
   - GPS 스마트 매칭 + 실시간 사진 전송
   - SNS 연결: 자녀 세대·직장인 → "병원 다녀온 내용, 가족도 바로 알 수 있어요"

3. 6단계 실시간 안심 서비스
   - 픽업 → 병원 도착 → 진료 중 → 처방약 → 귀가까지 전 과정 사진 증명
   - 예약 후 총 4회 확인 메시지 발송
   - SNS 연결: 1인가구·직장인 → "말이 아니라 기록으로 증명합니다"

4. 글로벌 역량 특화 매니저
   - 다국어 지원, 송도·영종 외국인 병원동행·의료관광 동행 특화
   - 예약·일정·결제·정산 통합 관리
   - SNS 연결: 외국인·기업복지 → "언어 장벽 없는 병원 동행"

5. 3중 보안 + 보험 연동
   - 이용자 정보 3중 보안 처리
   - 보험 연동 시스템으로 책임감 있는 업무 완수
   - SNS 연결: 모든 독자 공통 → "믿고 맡길 수 있는 이유가 있어요"


■ SNS 글 포맷

- 글자 수: 200~350자 (해시태그 제외)
- 이모지: 2~4개, 감정 흐름에 맞게 자연스럽게
- 해시태그: 7~10개, 글 맨 마지막 별도 줄

구조 (3단계):
[1] 공감 훅 (1~2줄)
    → "어, 나 얘기다" 하는 상황 묘사
    → 질문형 or 상황 묘사형
    → 지역명 포함 시 노출 효과 상승 (예: "송도에 사시는 분들")

[2] 문제 + 해결 실마리 (3~5줄)
    → 공감 확장 + 모두의병원동행 자연스럽게 소개
    → 차별점 1개를 이야기로 풀기
